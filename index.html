Role Descriptions
🔑 Admin
Full unrestricted access. Can manage users, change settings, clear data, and perform all operations. Always has all permissions.
🏢 Manager
Operational access. Can add items, perform stock movements, and view reports. Specific permissions can be restricted below.
👁 Viewer
Read-only by default. Can browse inventory and view reports. No write operations unless explicitly granted below.
⚠ If user accounts are disabled (no users registered), the system runs in open mode and all features are accessible to everyone.
